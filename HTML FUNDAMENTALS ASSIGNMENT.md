                 HTML FUNDAMENTALS ASSIGNMENT	
1)	<!DOCTYPE html> tells the browser which version of html the document is using, it must be written at the very top of the file so the browser knows how to properly display the webpage.
Without it, the browser can get confused and may display the webpage using old rules which can make the website look broken and messy.

2)	The <head> contains information about the web page that   is not usually shown on the webpage itself. It is used for things like page title, linking CSS files and java script files.

The <body> contains everything that the user actually sees on the web page. It includes headings, paragraphs, images, links and others.

3)	Using headings <h1> through to <h6> in the correct order helps organize a webpage clearly, just like chapters and subheadings in a book.
It helps browsers understand page structure, it helps visually impaired users to navigate page and it also helps search engines to understand your content better. 

4)	An html attribute is a word added to an HTML tag to give it more information or tell it what to do.
For images the attribute used is   <img src = “photo.jpg” alt = “A photo”>. Which tells the browser where the image is coming from.

For links, the attribute used is <a href=https://example.com> visit site </a>. which tells the browser where the link should take the user. 

5)	Inside a <p> tag the web browser automatically ignores the extra spaces and line break. Even if HTML code has many spaces of lines , the browser will display the text normally in one paragraph.
